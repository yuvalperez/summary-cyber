#מה-זה-cache
רגע לפני נסביר מה זה cache ,cache נועד כדאי להפוך את השרת ליותר מהר במקום השרת שכנראה רחוק יותר מהשתמש ועמוס יותר. עשו  cache שזה סוג של מיני שרת מהיר יותר שומר בו דברים סטטים לבקשות מהירות cache. הcache יושב בין השרת למשתמש  הcache קרוב יותר למשתמש = תגובה מהירה יותר. הcache חוסך עבודה לשרת שנותן מהירות למשתמש.

#מה-זה-CDNs 

CDN = Content Delivery Network



האתר  יושב על שרת אחד באמריקה. משתמש מישראל מבקש את האתר — הבקשה עוברת את האוקיינוס = איטי.

CDN פותר את זה על ידי שיש לו שרתים בכל העולם:

```
שרת בישראל
שרת באירופה  
שרת באמריקה
שרת באסיה
וכו...
```

המשתמש מישראל מקבל את התגובה מהשרת הקרוב אליו בישראל — מהיר מאוד.

הקשר ל cache:

CDN הוא בעצם cache ענקי מבוזר בכל העולם. כל שרת של ה CDN שומר עותקים של המשאבים הסטטיים של האתר.

דוגמאות לחברות CDN:

```
Cloudflare
Akamai
Amazon CloudFront
Fastly
```
